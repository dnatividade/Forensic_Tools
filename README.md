# Forensic Tools

### ADS ZoneID
```
Descubra de onde veio um arquivo baixado.
Funciona apenas em sistemas de arquivos NTFS.
```

### Quando Desligou Windows
```
Script em VBS que mostra quando foi a última vez que o WIndows foi desligado.
```
